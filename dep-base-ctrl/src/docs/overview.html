The test admin platform control is one of the new platform artifacts constructed as part of the OAS 4.0/5.0 re-architecture effort. It is intended to provide all data access and business logic facilities required by OAS application developers within its functional area. The control is provided as a jar file along with this documentation for use by application developers. The test admin control specifically provides methods for obtaining lists of test sessions of interest to a user, obtaining detailed information about those sessions, obtaining lists of session reports available to a user, and obtaining a list of active system broadcast messages. original release: v0.1, 11/22/2005, ncohen
